## Project Scope

This library is a personal quantitative finance sandbox focused on:
- Option pricing and implied volatility
- Curve construction (rates, forwards)
- Volatility surfaces and structures
- Model experimentation and education


## Planned Structure (WIP)

- `instrument/` – Options and underlying instruments
- `curve/` – Discount, forward, and rate curves
- `vol/` – Vol surfaces, smiles, and parameterizations
- `data/` – Market data adaptors (e.g. Yahoo Finance)
- `notebooks/` – Documented examples and experiments
